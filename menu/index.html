welcome to squiggle.city

This is a place for messing with unix, making web pages, and hanging out. It is inspired by tilde.club, but may have ideas of its own.

You're currently inside a text based web browser called Lynx. Use the arrow keys to navigate between links, and hit → or enter to follow a link. Hit "q" to return to the command line.

things you can do

  • Read your mail in Pine
  • Edit your home page
  • Edit your .plan
  • See who's logged in
  • Change your shell
  • Ask a pony how long the server has been up
  • Change your password (not needed for most things here)
  • Edit your SSH authorized key list (CAREFUL!)

games you can play

  • Adventure
  • Zork
  • NetHack
  • Hunt the Wumpus

(...there are more. Try doing ls /usr/games.)
